# AI Data Prep Coach 🚀

An AI-powered interview preparation platform designed for aspiring **Data Analysts, Data Scientists, Business Analysts, and Analytics Freshers**. This application helps users practice interview questions across core data science domains, receive instant evaluations, and improve both technical and communication skills through a structured learning approach.

🌐 **Live Demo:** [AI Data Prep Coach](https://hrishiyash.github.io/data-prep-coach/)

---

## 📌 Overview

Preparing for data-related interviews often involves navigating hundreds of questions from different sources. AI Data Prep Coach consolidates this process into a single platform where users can:

* Practice **65+ curated Data Science and Analytics interview questions**
* Progress through **Beginner, Intermediate, and Advanced levels**
* Receive **AI-generated answer evaluations**
* Understand strengths and improvement areas
* Build confidence for real-world interviews

The platform was developed using AI-assisted development workflows with **Claude** and **ChatGPT**, combining rapid prototyping, prompt engineering, and interview-focused content generation.

## ✨ Key Features

### 📚 Comprehensive Question Bank

Covers essential topics including:

* SQL
* Python
* Statistics
* Machine Learning
* Data Analysis
* Data Visualization
* Power BI
* Excel
* Business Analytics
* Data Science Fundamentals

### 🎯 Difficulty-Based Learning

Questions are categorized into:

* 🟢 Beginner
* 🟡 Intermediate
* 🔴 Advanced

This allows users to gradually build their knowledge and confidence.

### 🤖 AI-Powered Evaluation

For every answer submitted, the platform provides:

* Answer assessment
* Technical accuracy review
* Improvement suggestions
* Expected interview-quality response guidance

### 📈 Skill Development

Users can:

* Identify weak areas
* Track learning progress
* Improve problem-solving skills
* Enhance interview readiness

### 💬 Interactive Learning Experience

Instead of passive reading, users actively engage with questions and receive immediate feedback.
